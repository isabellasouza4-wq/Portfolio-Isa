Minha experiência com Git e GitHub

Nesta atividade, utilizei o Git para controlar as versões dos arquivos do meu portfólio e o GitHub para armazenar o repositório remoto.

Dificuldades encontradas

Uma das dificuldades foi entender a diferença entre Git e GitHub. Também tive dificuldade em usar os comandos pelo terminal.

Como resolvi

Resolvi pesquisando os comandos, seguindo o passo a passo da atividade e testando os comandos git status, git add, git commit e git push. Também pedindo ajuda para colegas e vendo vídeos de tutoriais sobre Git e GitHub.

Conclusão

Ao final, consegui criar uma nova página, adicionar ao repositório, fazer um commit e enviar as alterações para o GitHub.
